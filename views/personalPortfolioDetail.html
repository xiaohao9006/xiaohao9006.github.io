Grid

Grids divide horizontal space into indivisible units called "columns". All columns in a grid must specify their width as proportion of the total available row width. I chose an 16 column count to allow per row. And each website element span a certain number of columns.

A vertical rhythm was used to achieve visual consistency in website design.

Baseline Grid

No matter what font-size any text element is, its line-height is always an even multiple of a consistent unit of rhythm. I chose the base line-height as 20px, and base font-size as 14px. Then every website element's height, padding and margin is a multiple of base line height.

Using Angulartics to Track Single Page Application

According to Google Analytics Developers guide, since my website is wrapped as a single page application, simply including analytics.js snippet code into my website cannot work well, because the site loads new page content dynamically rather than as full page loads, and the analytics.js snippet code only runs once. This means subsequent (virtual) pageviews must be tracked manually as new content is loaded. Also event tracking has to be manually implemented as well. Therefore, Angulartics was used to achieve the level of sophistication of data collection.

/*Inject Angulartics in application dependencies*/
angular.module('projectApp',['angulartics', 'angulartics.google.analytics'])
/*Inject $analytics and invoke either the pageTrack() or eventTrack() methods to track pageviews and events.*/
angular.module('projectApp')
  .controller('SampleController', ['$analytics', function($analytics){
    $analytics.pageTrack('url');
    $analytics.eventTrack('eventName');
    //...
  }]);

Using Gulp to Minify the Website Resources

One of the ways that my audience will evaluate my website, especially on their first visit, is my page load speed. If my pages take too long to load, it is likely that audience will move along. Therefore, minification, removing unnecessary or redundant data without affecting how the resource is processed by the browser, is one important step before publishing.

When the environment was set up by Yeoman, minification was already included in Gulp build configuration file, which I directly leveraged to get this done.
